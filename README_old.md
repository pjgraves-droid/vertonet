# Vertonet Website

Marketing site for **Vertonet** — an AI Software Engineering services company bringing Cognition's Devin to the Australian and New Zealand market.

## Overview

A single-page static site built with vanilla HTML and CSS (DM Sans + JetBrains Mono from Google Fonts). No build step required — open `index.html` directly or serve with any static file server.

## Deployment

Deploy to any static hosting provider:

```bash
# Netlify, Vercel, Cloudflare Pages, etc.
# Just point the root to this directory.
```

## Structure

```
vertonet-site/
├── index.html   # Complete single-page site

```

## Sections

- **Hero** — Value proposition and CTAs
- **Why Vertonet** — Three-column value grid (Ship faster, Scale, Security)
- **Services** — Pilot Program, Enterprise Rollout, Technical Consulting, Managed AI Engineering
- **Partnership** — Cognition/Devin partnership details
- **Market** — Australian market positioning (AEST, AUD, SOC 2, On-site)
- **Contact CTA** — Email link to hello@vertonet.com.au
